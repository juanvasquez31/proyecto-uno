### Que vamos a hacer hoy?
* Manejo basico de GIT
    * Clonamos un proyecto
* Manejo basico de la consola (bash)
    * Crear carpeta: mkdir
    * Donde estamos: pwd
    * Ingresar a la carpeta: cd
    * Listar carpeta: ls
    * Eliminar carpeta: rm -rf
* Markdown Basico
* Manejo basico de archivo 
    * Abrir un archivo
    * Leer el contenido 
    * Modificar el contenido
    * Cerrar el archivo
